fputc
----
向文件中写入字符
###格式
    fputc(ch,fp);
###返回值
返回刚写入到文件中的字符，若出现错误则返回EOF。
###示例
```c
...
for(i=0;i<10;i++)
{
    ch=getchar();
    fputc(ch,fp);
}
```
fgetc
-----
从文件读取单个字符
###格式
    ch=fgetc(fp);
###返回值
返回读到的字符，并转型成int。若到文件结尾返回EOF
###示例
```c
...
for(i=0;i<10;i++)
{
    ch=fgetc(fp);
    putchar(ch);
}
```
