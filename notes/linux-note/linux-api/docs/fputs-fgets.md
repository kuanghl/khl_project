fputs
-----
写入字符串到文件
###函数原型
```c
#include <stdio.h>
int fputs(const char *s,FILE *fp);
```
###功能描述
将s指向的字符串，舍去结束标记'\0'后写入到fp指向的文件缓冲区。
###返回值
返回写入的实际字符数；出现错误，返回EOF。

fgets
-----
从文件读取字符串
###函数原型
```c
char *fgets(char *s,int size,FILE *fp);
```
###功能描述
从fp所指向的文件缓冲区当前位置读取n-1个字符，
在其后补充一个'\0'，并写入到s指向的内存区。
###返回值
返回s对应的地址，可以理解为返回字符串。出现错误返回NULL。
